Screenshot Webservice

An open-source REST web service to perform web page screenshot powered by Play! framework and PhantomJS.

Try it out

  • GET a screenshot of github.com: /screenshot.png?url=http://github.com/&size=1024x1024
  • process it with a HEAD: curl -I http://localhost:9000/screenshot.png?url=http://github.com/
